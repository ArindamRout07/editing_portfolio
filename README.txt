# 🎬 Arindam Rout – Portfolio Website

A modern, cinematic portfolio website showcasing my work as a **Video Editor** and **Creative Developer**. Built with React and Tailwind CSS, the website highlights my editing style, featured projects, creative workflow, and services through an immersive user experience.

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

Go inside the project

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build for production

```bash
npm run build
```
