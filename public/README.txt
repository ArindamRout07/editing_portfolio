Place the following files inside this `public/` folder before deploying to Vercel:

  - hero.mp4                          (hero background video)
  - profile.jpg                       (photo used in the About section)
  - midnight-tokyo-first-frame.jpg    (thumbnail for project #1)

Vite copies the entire contents of `public/` into the build root, so files
in here are served from the site root, e.g. /hero.mp4, /profile.jpg.
